# Dasbor Cuaca Pegunungan

Aplikasi React + Vite ini mengambil data cuaca real-time dari OpenWeatherMap.

## Fitur

- Pencarian lokasi atau gunung
- Menampilkan suhu saat ini, kelembapan, kecepatan angin
- Prakiraan cuaca 5 hari
- Peringatan visual saat ada hujan lebat / badai
- Loading state dan error handling

## Jalankan

1. Install dependensi:

```bash
npm install
```

2. Buat file `.env` di dalam folder `weather`:

```env
VITE_OPENWEATHER_API_KEY=your_openweather_api_key_here
```

3. Jalankan aplikasi:

```bash
npm run dev
```
